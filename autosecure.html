  • /claim <ign> - Claims a Minecraft account
  • /dm <user> <message> - Messages a user anything
  • /embed <type> - Sends the Phisher or OAuth Embed
  • /getssid <logincookie> - Gets account SSID with Microsoft login cookie
  • /guide - Sends a guide to log in with the login cookie
  • /profiles - Panel to configure different profiles
  • /requestotp <email> - Sends an OTP code to a security email
  • /secure <type> - Secures an account with recovery code or OTP code
  • /set <channel> - Sets the channels for autosecure
  • /settings - Sends the settings panel to configure everything
  • /ssidchecker <ssid> - Checks a SSID
  • /stats <ign> - Gets the Hypixel stats of an account
  • /users - Sends the user panel to configure admins and claiming
